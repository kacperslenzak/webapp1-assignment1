# Movie Tracker App

👋 This is a movie tracker app that allows for tracking the movies you have watched!

## What it does
- Shows a dashboard with all the movies in your catalogue
- Lets you view details about each movie
- Has an about page with info
- Stores data using JSON files

## Features
- **Dashboard**: See all movies as cards in a grid
- **Movie Pages**: Click to see more about a movie
- **About Page**: Learn about the app
- **Responsive Design**: Looks good on phones and computers thanks to Fomantic UI

## How to run it
1. Make sure you have Node.js installed
2. Download or clone this project
3. Open a terminal in the project folder
4. Run `npm install` to get all the dependencies
5. Run `npm start` to start the server
6. Open your browser and go to `http://localhost:3000`

## Tech stack
- **Node.js**: For the backend server
- **Express**: To handle routes and stuff
- **Handlebars**: For making the HTML templates
- **Fomantic UI**: For the nice styling
- **LowDB**: For storing data in JSON files
- **Winston**: For logging
